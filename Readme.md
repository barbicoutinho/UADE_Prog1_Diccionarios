# Proyecto Diccionarios – Gestión de Carrito y Stock

Este proyecto simula el funcionamiento básico de un sistema de ventas donde se cargan artículos en un carrito y se verifica si existe stock suficiente para procesar un pedido.

## Estructura del proyecto

- `main.py`: archivo principal.
- `functions/`: contiene las funciones de verificación de stock y procesamiento de pedidos.
- `utils/`: incluye validaciones y manejo de archivos.
- `data/`: almacena los diccionarios reales en formato JSON.
